Database Migration Service; DMS

  • 用在將On-Primse DB轉移到AWS的服務
  • 轉移期間資料庫可以正常使用
    • 可以停機，一次性轉移；或不停機，持續複寫
  • 遷移目標可以是RDS或EC2
    • 以DB的FQDN為目標，不是以subnet或VPC為目標
  • 當來源與目標的schema不同時
    • 設定Transformation Rule進行轉換
    • 可以做到的
      • 僅遷移特定範圍的資料
      • 跳過具有特定pattern的資料(例如test開頭的紀錄)
      • Schema mapping，某個column改名叫另一個column名稱
      • 對table, column加上prefix
  • 支援同質/異質轉移
    • 異質轉移：透過免費的AWS Schema Conversion Tool (SCT)，轉換PL/SQL, T-SQL與MySQL SQL
    • 若無法自動轉換，會標記出有問題的位置給開發者
